# OutsOpsSocialSituation

Materials for the study on social situations hosted by children helping science.
This repository contains the auditory stimuli, visual materials, and documentation used in the study.

---

## Repository structure

- [`audios/`](audios/)  
  Audio files used in the experimental tasks (e.g., instructions, stimuli).

- [`images/`](images/)  
  Visual materials, figures, and stimuli used in the tasks and documentation.

---

## Using this repository

You can access the folders directly on GitHub:

- Browse audio stimuli: [`audios/`](audios/)
- Browse visual stimuli: [`images/`](images/)

Or, clone the repository locally:

```bash
git clone https://github.com/virginiamezzalira/OutsOpsSocialSituation.git
cd OutsOpsSocialSituation
